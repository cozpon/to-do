# React To-Do list

## Getting Started:

### Clone to your local computer:
in a terminal window run:
```
git clone https://github.com/cozpon/to-do
```
cd into 'to-do' directory and run:

```
npm install
```
to install the node_modules


Then run:
```
npm start
```
Open your browser and go to:
```
http://localhost:8080/
```


### Enjoy!
